Autonomous Golf Cart

Timeline: October 2015 - May 2016

Contact: Joshua Price - jprice [at] berkeley.edu

Description:

We love autonomous vehicles! This project group is equipping a golf cart with the mechanical and electrical equipment necessary to control it via electrical signals. They will be using vision processing and advanced sensors to map the vehicle's surroundings so it can be programmed to avoid obstacles and follow paths. If you see a golf cart driving itself around campus, it's probably the work of this project group!

This project is currently accepting more members, so fill out this form if you are interested in joining and Josh will get in contact with you.
